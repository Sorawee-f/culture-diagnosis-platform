# Prototype v3 UI Update

Version: `2026-pilot-v3-ui`

## สิ่งที่ปรับ
- เพิ่ม Dimension ภาษาไทยเป็นป้ายเล็กเหนือหัวข้อสถานการณ์
- ใช้โทนน้ำเงินสำหรับ “สิ่งที่เป็นอยู่ในปัจจุบัน”
- ใช้โทนเขียวสำหรับ “อนาคตที่อยากเห็น”
- แบ่งสองมุมมองด้วยเส้นและข้อความ “แล้วจึงตอบ”
- เพิ่มหมายเลขขั้นตอน 1 และ 2
- แสดงสถานะ “เลือกแล้ว / ยังไม่ได้เลือก” แยกแต่ละฝั่ง
- แสดงสรุปว่าตอบแล้วกี่มุมมอง
- ปุ่มถัดไปใช้งานได้เมื่อเลือกครบทั้งสองฝั่ง
- Mobile เรียงบน–ล่าง และ Desktop แสดงสองคอลัมน์

## Data / Scoring
- Scenario ID, Option ID และ Archetype Mapping ไม่เปลี่ยน
- ไม่ต้องแก้ Supabase Schema
- เนื่องจากการเพิ่ม Dimension และเปลี่ยน Presentation อาจมีผลต่อการรับรู้ของผู้ตอบ แนะนำแยก Pilot ด้วย `SURVEY_VERSION=2026-pilot-v3-ui`
