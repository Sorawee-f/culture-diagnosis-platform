# Culture Survey Instrument Selection Pilot

## Objective

เปรียบเทียบแบบสำรวจ 2 ชุดเพื่อใช้เป็นข้อมูลประกอบการเลือกแบบสำรวจสุดท้าย โดยผู้เข้าร่วมมีหน้าที่ทดลองระบบและตอบแบบสำรวจเท่านั้น

## Test Design

ผู้ทดลอง HR ทุกคนทำทั้ง 2 ชุด ระบบสลับลำดับอัตโนมัติ

| Group | First | Second |
|---|---|---|
| A | Scenario-Based | Simplified |
| B | Simplified | Scenario-Based |

ทั้งสองชุดใช้ Side-by-Side เพื่อควบคุมเงื่อนไข

หลังทำครบ ผู้ทดลองจะลอง Mini Interactive Demo ของ Side-by-Side และ Sequential เพื่อทดสอบความชัดเจนของหน้าจอ

## Evidence from System

1. Completion Count และ Completion Rate
2. Average Completion Time
3. Current / Desired Culture Profile
4. Top Archetype ของแต่ละชุด
5. Culture Gap Direction
6. Mean Absolute Difference ระหว่างสองชุด
7. ผล Survey Mode Test
8. ข้อสังเกตด้านการใช้งานผ่านมือถือ

## Interpretation

คะแนนไม่จำเป็นต้องตรงกันทุกเปอร์เซ็นต์ ให้ดูว่า:

- ทิศทาง Culture Profile สอดคล้องกันหรือไม่
- Current Top และ Desired Top เหมือนกันหรือไม่
- Direction ของ Culture Gap เหมือนกันกี่ Archetypes
- แบบง่ายเปลี่ยนข้อสรุปเชิงทิศทางหรือไม่
- หน้าจอบนมือถือทำให้ผู้ตอบทำครบได้สะดวกหรือไม่

## Decision Owner

ผู้ทดสอบไม่ต้องเลือกชุดคำถาม ทีมโครงการจะนำข้อมูลจาก Dashboard ไปประกอบการพิจารณาร่วมกับหัวหน้าและ Co-CEO
